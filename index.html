이 홈페이지는 index.html 클릭만으로 열리며, 나중에 GitHub Pages에 업로드해도 사용할 수 있도록 구성했습니다. 공개 사료는 출처와 함께 표기하고, 직계 정보와 문중 사진은 추후 계속 추가할 수 있게 설계했습니다.

사진·영상 아카이브

GitHub Pages 업로드용 자동 배열 방식을 넣었습니다. GitHub에 올린 뒤 media 폴더에 사진과 영상을 넣으면 자동으로 읽어와 정렬해서 보여줍니다. 로컬에서 index.html만 실행할 때는 media.json 목록을 참조합니다.

자동 배열 방식

  • GitHub 업로드 시: media/ 폴더 자동 조회
  • 로컬 실행 시: media.json 목록 사용
  • 이미지와 동영상이 날짜순·이름순으로 배열

권장 파일명 예시

  • 2024-04-14_묘사1.jpg
  • 2024-04-14_재실1.jpg
  • 2024-04-14_묘사봉행.mp4

비용관리 안내

이 버전은 브라우저 저장(localStorage) 방식입니다. 계좌, 장부, 영수증 설명이 저장되며 같은 브라우저에서 다시 확인할 수 있습니다. 통장 이미지에서 거래내역을 자동으로 추출하는 OCR 기능은 아직 연결하지 않았습니다.
