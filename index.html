홈페이지 첫 화면에는 요청에 따라 26대까지의 흐름만 요약 표기합니다. 정담 이후 중간 세대는 연구 구간으로 두고, 26대는 세보 대조상 가장 유력한 인물로 정리했습니다.

ADD RECORD

후손 기록 추가

브라우저 저장 방식입니다. 같은 PC에서 다시 열면 저장된 후손 정보가 유지됩니다.

MEDIA ARCHIVE

사진·영상 아카이브

관리자는 휴대폰이나 PC에서 사진·영상을 선택하여 순서번호를 입력하고, 기존 항목의 삭제 및 순서 수정이 가능합니다. 정적 홈페이지 특성상 브라우저가 서버의 media 폴더에 직접 저장하지는 못하므로, 관리자 화면에서 목록을 정리한 뒤 media.json을 내려받아 기존 파일과 함께 업로드하면 반영됩니다.
MEDIA ADMIN

관리자 사진 관리

휴대폰과 PC 모두 파일 선택 후 등록할 수 있습니다. 반영 시에는 내려받은 media.json과 실제 사진 파일을 함께 media 폴더에 올려 주세요.

관리 기능

  • 순서번호 수정
  • 제목 수정
  • 기존 항목 삭제
  • 정렬 결과 즉시 미리보기
