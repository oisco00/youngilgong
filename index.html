비용관리 안내

이 버전은 브라우저 내부 저장(localStorage) 방식입니다. 영수증 이미지는 바로 올려 보관할 수 있고, 장부는 수기 입력 또는 CSV 업로드로 관리할 수 있습니다.
다만 통장 이미지에서 자동으로 거래내역을 읽는 OCR 기능은 아직 연결하지 않았습니다. 정확한 자동 인식을 원하면 향후 OCR/API 연동 버전으로 확장하면 됩니다.

문중 사진 · 영상 열람

같은 폴더에 있는 사진과 동영상을 보기 쉽도록 업그레이드한 기능입니다.
브라우저 보안상 폴더 내용을 완전히 자동으로 읽는 것은 제한되므로, 아래의 폴더 선택 또는 파일 선택 버튼을 한 번 눌러 주시면 현재 폴더 또는 원하는 폴더의 이미지·동영상을 바로 갤러리로 표시합니다.
권장 방식: 본 사이트 파일과 사진·영상 파일을 같은 폴더에 두고 폴더 선택을 누른 뒤 해당 폴더를 지정하세요.

갤러리

출처·사료 링크

아래 링크는 공개 웹에서 확인한 기본 사료·개요 자료입니다. 문중 홈페이지 본문 중 공개 검증 부분은 이 자료를 바탕으로 초기 구성했습니다. 추후 『영일정씨 좌랑공파보』 원문, 문중 소장 사진, 재실 편액, 제향 기록, 족보 스캔본을 추가하면 훨씬 완성도가 높아집니다.
본 사이트는 index.html 클릭으로 바로 실행되도록 제작된 정적 웹사이트입니다. 충분히 보완된 뒤 GitHub Pages에 업로드해 운영할 수 있습니다.
