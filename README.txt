영일정씨 문중 홈페이지 최종본 사용안내

1. index.html을 더블클릭하면 홈페이지가 실행됩니다.
2. 사진·영상을 보려면 상단 메뉴의 "사진·영상"으로 이동하세요.
3. 같은 폴더의 사진·영상까지 보려면:
   - 권장: "폴더 선택하여 불러오기" 버튼 클릭 후 현재 폴더 선택
   - 대안: "파일 여러 개 선택" 버튼으로 사진/영상 직접 선택
4. 비용관리 데이터와 후손 등록 데이터는 현재 브라우저(localStorage)에 저장됩니다.
5. GitHub Pages에 올릴 때는 media 폴더를 별도로 두고 파일명을 정리하면 운영이 더 편리합니다.

참고:
- 브라우저 보안상 완전 자동 폴더 스캔은 제한됩니다.
- Chrome/Edge 계열 브라우저에서는 "폴더 선택" 기능이 가장 잘 동작합니다.
- 묘사 사진/영상, 재실 사진, 편액, 벌초, 행사 사진 등을 계속 누적할 수 있습니다.
