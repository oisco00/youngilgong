영일정씨 문중 홈페이지 최종본 사용 안내

1. 바로 실행
- 압축 해제 후 index.html을 더블클릭하면 실행됩니다.
- media 폴더는 함께 포함되어 있으니 사진과 영상을 그 안에 넣어 사용하세요.
- 로컬에서 바로 열 때는 media.json에 적힌 목록이 먼저 표시됩니다.

2. GitHub Pages에 다시 올리는 방법
- 압축을 푼 폴더 전체를 저장소에 업로드하면 됩니다.
- 기본 GitHub 아이디는 oisco00으로 넣어 두었습니다.
- 저장소 이름(repo)은 비워 두어도 GitHub Pages 주소에서 자동 감지되도록 구성했습니다.
- 별도 수정 없이도 일반적인 GitHub Pages 구조에서는 media 폴더를 자동 조회합니다.

3. 사진·영상 올리는 방법
- media 폴더에 JPG, PNG, WEBP, MP4 파일을 넣으세요.
- 로컬에서는 media.json에 파일명을 추가하면 즉시 표시됩니다.
- GitHub Pages에서는 media 폴더 안 파일을 자동으로 정렬해 표시합니다.
- 권장 파일명: 2024-04-14_묘사1.jpg

4. 비용관리
- 계좌, 장부, 영수증 설명은 브라우저 저장 방식입니다.
- 같은 브라우저에서는 유지되지만 다른 PC로 옮길 때는 데이터 내보내기(JSON) 기능을 사용하세요.
